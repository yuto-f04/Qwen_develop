# 卒業研究メモ：秘密のキャンパスツアー音声生成

## プロジェクト概要

Qwen3-TTS で実際の人物の声をクローンし、キャンパスツアーガイドの台本を自動読み上げする。
「70〜90%が真実・10〜30%が嘘」という構成で、参加者が「どこまで本当か？」と楽しめるツアー音声を生成する。

---

## 全体フロー

```
YouTube動画
  → yt-dlp でMP3ダウンロード
  → Demucs でBGM・ノイズ除去
  → clean_audio1.wav（参照音声 ← 完成済み）
  → [Cell③] Whisper で文字起こし → REF_TEXT
  → 参照音声が30秒超なら自動カット → TRIMMED_AUDIO
  → [Cell⑤] Qwen3-TTS で台本を文単位に分割して生成
  → 文の区切りごとに無音を挿入
  → tour_guide.wav として出力
  → デモンストレーションツアーで使用
```

---

## 技術スタック

| 役割 | ツール |
|------|--------|
| 音声クローン生成 | Qwen3-TTS-12Hz-1.7B-Base |
| 参照音声の文字起こし | OpenAI Whisper (base モデル) |
| 音声の前処理（BGM除去） | Demucs（実施済み） |
| 実行環境 | Google Colab (GPU: L4 推奨) |

---

## ノートブックの構成（5セル）

| セル | 内容 |
|------|------|
| ① | ライブラリ一括インストール |
| ② | Googleドライブのマウント |
| ③ | 参照音声の長さチェック → 自動カット → **Whisperで文字起こし（REF_TEXT）** |
| ④ | Qwen3-TTSモデルのロード |
| ⑤ | 台本を文分割 → 1文ずつ生成 → **句点後に無音挿入** → 結合 → tour_guide.wav |

> ③の文字起こし（REF_TEXT）は⑤の `generate_voice_clone(ref_text=REF_TEXT)` に渡される。
> ref_audio と ref_text が一致していることが生成品質の鍵。

## イントネーション問題

**「しませんか？」などの疑問文で語尾が上がらない。**

現状の Qwen3-TTS ではイントネーション（抑揚）の細かい制御は困難。考えられる対策：
1. フリー音声素材（疑問形が多く含まれるもの）を参照音声として試す
2. 参照音声の学習時間を長くする（ただし下記の長さ問題あり）
3. 台本側で疑問形を減らし、断言形に書き換える

---

## 参照音声の長さ問題

- 参照音声が長すぎる（例: 複数動画を結合して9分）と生成できない
- 現在の方針: **長さではなく上限内で質を上げる**（MAX_REF_SECONDS = 30）
- Whisperの文字起こしもカット後の音声に対して実行するため、ref_text と ref_audio が一致する

---

## 嘘の設計ガイドライン

### 人が嘘だと感じやすい特徴
- 抽象的・極端な言葉（「完全な」「絶対に」）
- 話のつじつまが合っていない
- スケールがぶっ飛んでいる
- 声のトーン・間合いが不自然（→ 音声品質の問題でもある）

### 人が嘘だと判断しにくい特徴
- 具体的な数値やデータ（「280席」「幅4m・長さ100m」など）
- 実在する企業名や本当の事実が一部混ざっている
- 理屈が通っていて現実的にありそう
- スケールが現実的で、つじつまが合っている
- 「最新」「国内初」などのトレンドワードを含む

### 目安の構成比
**真実 70〜90% ／ 嘘 10〜30%** → 楽しめるツアーになる

## 訪問スポットと台本の構成

1. **H121 シアター教室（1F）**
   - 真実: JBLスピーカー・BARCO製4Kプロジェクター・280席・映画館仕様遮音
   - 嘘: 特定低周波で脳波強制覚醒する「居眠り防止実験」

2. **KOBO（H棟1F）**
   - 真実: 3Dプリンター・企業連携・地域ワークショップ・利用時間制限
   - 嘘: プラスチックゴミを分子レベルで分解する「極小リサイクルプラント」

3. **イノベーションラウンジ（4〜9F吹き抜け）**
   - 真実: 視覚的連続性・照明施設賞・学びの階段・デモストリート
   - 嘘: 地下サーバーの排熱を使う「全館循環型巨大煙突」

## 既知の問題と対処

| 問題 | 対処 |
|------|------|
| 参照音声が長くてエラー | MAX_REF_SECONDS = 30 で自動カット。それでもエラーなら 15 に変更 |
| 文の区切りに不自然な空白 | 句点後の無音挿入（400ms / 300ms）で対応済み |
| 台本が長くてメモリ不足 | 1文ずつ生成して最後に結合 |
| 生成が遅い・終わらない | ColabのランタイムをL4 GPUに変更 |
| Whisper文字起こしが不正確 | 手動で修正するか、Whisper largeモデルを使う |
